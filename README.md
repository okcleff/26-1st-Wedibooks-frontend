## 구현 기능
> 회원가입 / 로그인 기능 구현
- 회원가입, 로그인 유효성 검사
- 메인페이지 (추천도서, 베스트셀러, 신간도서) 슬라이드 기능
- 카테고리별 베스트셀러 페이지 라우팅
- 상세페이지 게시글, 좋아요, 좋아요 취소, 별점기능
- 장바구니 페이지 가격 합산 기능
